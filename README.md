## Install

```sh
npm install --save @datastructures-js/stack
```

- Bubble Sort O(n<sup>2</sup>)
- Selection Sort O(n<sup>2</sup>)
- Insertion Sort O(n<sup>2</sup>)
- Radix Sort (numbers only) O(n\*k)
- Heap Sort O(n\*log(n))
- Quick Sort O(n\*log(n))
- Merge Sort O(n\*log(n))
